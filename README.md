# Time Tracker

Track work hours by entering start/end times or hours directly.

## Setup

```bash
npm install
npm start
```

Open http://localhost:3000

## Usage

1. Enter task name
2. Set times or toggle to enter hours
3. Click "Add Entry"

## Notes

- Data resets on page refresh (no saving)
- Time format is 24-hour (18:30 not 6:30 PM)
